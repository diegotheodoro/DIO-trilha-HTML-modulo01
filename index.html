História dos computadores

(Voltar)

Os primeiros computadores foram criados na década de 1940, com o Mark I, seguido pelo Colossus e ENIAC, utilizados para cálculos durante a segunda guerra mundial, eram muito grandes por causa dos componentes utilizados.
Com a evolução das tecnologias foram desenvolvidos transistores, capacitores, resistores, os circuitos integrados fazendo com que os componentes utilizados nos computadores fossem diminuindo e por consequência eles também foram sendo compactados.

O que são Clients e Servers?

(Voltar)

O Client ou Cliente é que realiza a comunicação com o servidor, quando um site é acessado através de um navegador é enviada uma requisição por meio da internet para o computador está hospedado e se aguarda uma resposta do servidor.
Já os Servers ou Servidores são computadores robustos utilizados para darem apoio a outros, transferindo, armazenando arquivos e informações, atendendo as requisições que esses computadores solicitam.
Existem vários tipos de servidores como os de Arquivos, Segurança, Streaming, E-mail, Web.
Os servidores podem ser desde computadores pequenos utilizados em casa, como pode ser enormes que são os chamados Data Centers por exemplo os do Facebook e Google.

O que e Linguagem de Programação?

(Voltar)

Linguagem de Programação os conjuntos de informações inseridas no computador que são executadas de acordo com a lógica implementada.
Entre as linguagens mais populares utilizadas estão: Javascript, Python, Java, PHP, C#, GO.
A linguagem de programação é uma ferramenta para resolver um problema, quem resolve o problema é o programador.

O que é HTML?

(Voltar)

O HTML significa HiperText Markup Language ou Linguagem de Marcação de Hipertexto.
É utilizado para programação web, onde permite inserir o conteúdo e estabelecer a estrutura de um site ou software web.

Estrutura básica do HTML?

(Voltar)

A estrutura básica do HTML consiste em declarações de TAG´s de abertura e fechamento por exemplo: <html> e </html>, seguindo da tag <head> onde são inseridas as informações básicas da página título, links, metadados, entre outros, por meio da tag <title> é declarado o título da página.
Por fim é realizada a abertura da tag <body> </body> para que os elementos como: tabelas, formulários, imagens, entre outros elementos sejam inseridos e o corpo da página tome forma.
